\section[word-list]{Dictionary}


bosonic variables --- a commuting variable, $x_ix_j = x_j x_i$. 
See also \emph{fermionic variables}.

character


cluster algebra

Ehrhart polynomial

fermionic variables --- anti-commuting variables, $\theta_i \theta_j= - \theta_j \theta_i$.
Note that $\theta_i \theta_i = - \theta_i \theta_i$, so $\theta_i^2 =0$. 
See also \emph{bosonic variables}.
